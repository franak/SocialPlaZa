--- A. Installing the favicon ---

1. Upload favicon.ico to the root of your web folder.

2. Add the following HTML code to your web page, in between <head> and </head> tags:

   <link rel="shortcut icon" href="favicon.ico" >



--- B. Installing iPhone / iPod Touch icon ---

1. Upload apple-touch-icon.png to the root of your web folder.

2. Add the following HTML code to your web page, in between <head> and </head> tags:

   <link rel="apple-touch-icon" href="apple-touch-icon.png" >



--- C. What's in this favicon.ico ---

The favicon.ico in this package has 3 image formats in a single file: 
    16*16-pixel 256-color, 32*32-pixel 256-color and 48*48-pixel 256-color.
